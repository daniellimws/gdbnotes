#scripting 
GDB provides a simple yet powerful scripting interface that is very helpful while doing reverse engineering. A GDB script file can contain a collection of commands, used with the following constructs. Usually I call it GDB script, but apparently GDB docs calls them command files. source <file> The file name/extension does not matter. A simple GDB script looks like this. gdb gdbscript set $sum=0 break *foo commands silent printf "Arg: %d\n", $rdi set $sum=$sum+$rdi c end printf "Sum: %d\n", $sum To execute the script, simply call source .
