c basic.c // gcc -g basic.c -o basic -no-pie #include <stdio.h> int main() { int a = 10; printf("%d\n", a); return 0; } break To inspect the memory contents of the running process, we need to set a breakpoint. break *<function> To set a breakpoint at the start of main, call break *main.
